## 📌 プロジェクト概要

**商品名**: 越中舟橋駅の車内メロディクリエイト体験  
**キャッチコピー**: あなたの作ったメロディが、電車で流れる。

**販売数量**: 限定5点  
**目標CV**: 購入申込（TOKKENプラットフォームへの遷移）

### タイポグラフィ

#### フォント推奨案

**和文**
- 見出し: **Noto Sans JP (Bold/Black)** または **源ノ角ゴシック**
  - モダンで力強く、鉄道の堅牢性を表現
- 本文: **Noto Sans JP (Regular/Medium)** 
  - 可読性が高く、長文でも疲れない

**欧文・数字**
- アクセント: **Montserrat (Bold)** または **Raleway (Bold)**
  - 価格表示、数字の強調に

#### サイズ階層
```
H1 (FVキャッチ): 48px〜64px (SP: 32px〜40px)
H2 (セクション見出し): 36px〜42px (SP: 28px〜32px)
H3 (小見出し): 24px〜28px (SP: 20px〜24px)
本文: 16px〜18px (SP: 14px〜16px)
小文字: 12px〜14px (SP: 11px〜13px)
```

### アニメーション設計

**コンセプト**: "電車の到着を待つワクワク感" を表現

#### 実装するアニメーション
1. **スクロールトリガー型**
   - セクションのフェードイン（下から上へ）
   - 電車の走行イメージ（左から右へスライド）
   - 数字のカウントアップ（価格、限定数）

2. **ホバーエフェクト**
   - CTAボタン: 色の明度変化 + わずかな拡大
   - カード: 浮き上がるエフェクト（shadow変化）

3. **背景要素**
   - 線路のラインアニメーション（SVG）
   - 音符のパーティクル（軽微に）

4. **パララックス**
   - FV背景の緩やかな移動
   - セクション間の奥行き表現

## 📐 セクション構成

### 1. FV (First View)
**目的**: 一瞬で心を掴む

**要素**
- キャッチコピー（大）: 「あなたの作ったメロディが、電車で流れる。」
- サブコピー: 「富山地方鉄道・越中舟橋駅の車内メロディを作れる、一生に一度のクリエイト体験」
- **背景画像**: `01.jpg/png` - FV用画像（電車の車内または駅の雰囲気）
- CTAボタン: 「この体験を購入する」（山吹色）
- スクロール促進アイコン

**デザインイメージ**
```
┌──────────────────────────────────────┐
│                                      │
│   あなたの作ったメロディが、         │
│          電車で流れる。              │
│                                      │
│  富山地方鉄道・越中舟橋駅の          │
│  「車内メロディ」を作れる、          │
│   一生に一度のクリエイト体験。       │
│                                      │
│    [この体験を購入する]              │
│          ↓                           │
└──────────────────────────────────────┘
```

---

### 2. コンセプトセクション
**見出し**: 「あの駅の音」の作者は、あなたです。

**コンテンツ**
- 3つの訴求ポイント
  1. **鉄道ファンの夢**: 「もし、自分がこの駅のメロディを作れたら」
  2. **音楽クリエイターの実績**: 毎日走る電車に、あなたの音が組み込まれる
  3. **自由な表現**: 20秒に何を込めるかは、あなた次第

**レイアウト**: 3カラムまたはカードレイアウト（アイコン + テキスト）

---

### 3. ストーリーセクション
**見出し**: 鉄道を守り、地域とつながる新しい形

**コンテンツ**
- 富山地方鉄道の背景
- 地域課題への共感
- この企画の意義

**画像**: `03.jpg/png` - 新しい試みとわかる画像（鉄道×地域×クリエイティブの融合を表現）

**デザイン**: テキストと画像の2カラムレイアウト、または画像を背景に薄く配置してテキストをオーバーレイ

---

### 4. 体験内容・詳細セクション
**見出し**: 体験内容とステップ

**主要情報（ボックス表示）**
```
┌─────────────────────────────┐
│ 販売価格: 250,000円（税込）  │
│ 数量: 限定5点               │
│ メロディ仕様: 20秒以内      │
│ 掲載期間: 4ヶ月間           │
└─────────────────────────────┘
```

**放送路線リスト**
- アコーディオンまたはドロップダウン形式で13路線を表示

**視覚的工夫**: 
- 価格は大きく強調
- 「限定5点」を赤系または山吹色でハイライト

---

### 5. 特典セクション
**見出し**: 購入者限定特典

**コンテンツ**
- 非公開車庫写真のNFT権利証
- NFTの価値（コレクション性、希少性）

**画像**: `04.jpg/png` - 記念の権利証（NFTカードまたは権利証のビジュアル）

**デザイン**: カード型で高級感を演出、画像を中央に大きく配置

---

### 6. 制作フローセクション
**見出し**: 音源制作のステップ

**ステップ表示（4段階）**
1. **購入**: TOKKENプラットフォームにて購入
2. **メロディ制作**: 2ヶ月以内に制作・提出
3. **データチェック**: 事務局が確認
4. **掲載スタート**: 4ヶ月間の運行

**画像**: `02.jpg/png` - 納品ステップ（制作フローの全体像を示すビジュアル）

**デザイン**: 
- 画像を上部または背景に配置
- タイムライン形式または番号付きカードでステップを表示
- 各ステップにアイコン配置
- 電車が進む演出（アニメーション）

---

### 7. CTA（最終）
**見出し**: 越中舟橋駅の車内メロディクリエイト体験を購入する

**要素**
- 大きなCTAボタン（山吹色）
- ボタン上に「限定5点」を再強調
- 利用規約・注意事項へのリンク

**デザイン**: 
- セクション全体を緑の背景で囲む
- ボタンを中央配置、最大級のサイズに
